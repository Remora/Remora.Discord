ModifyThreadChannelAsync(Snowflake, Optional<string>, Optional<bool>, Optional<TimeSpan>, Optional<bool>, Optional<int?>, CancellationToken) Method

Summary

Modifies the given channel.
Namespace
Remora.Discord.Rest.API
Containing Type
DiscordRestChannelAPI

Syntax

public Task<Result<IChannel>> ModifyThreadChannelAsync(Snowflake channelID, Optional<string> name = default(Optional<string>), Optional<bool> isArchived = default(Optional<bool>), Optional<TimeSpan> autoArchiveDuration = default(Optional<TimeSpan>), Optional<bool> isLocked = default(Optional<bool>), Optional<int?> rateLimitPerUser = default(Optional<int?>), CancellationToken ct = default(CancellationToken))

Parameters

Name Type Description
channelID Snowflake The ID of the channel.
name Optional<string> The new name of the channel.
isArchived Optional<bool> Whether the thread is archived.
autoArchiveDuration Optional<TimeSpan> The time of inactivity after which the thread is archived.
isLocked Optional<bool> Whether the thread is locked.
rateLimitPerUser Optional<int?> The new rate limit per user.
ct CancellationToken The cancellation token for this operation.

Return Value

Type Description
Task<Result<IChannel>> A modification result which may or may not have succeeded.
