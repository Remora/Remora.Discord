AddGuildMemberAsync(Snowflake, Snowflake, string, Optional<string>, Optional<IReadOnlyList<Snowflake>>, Optional<bool>, Optional<bool>, CancellationToken) Method

Summary

Adds a user to the guild using their OAuth2 access token.
Namespace
Remora.Discord.API.Abstractions.Rest
Containing Type
IDiscordRestGuildAPI

Syntax

Task<Result<IGuildMember?>> AddGuildMemberAsync(Snowflake guildID, Snowflake userID, string accessToken, Optional<string> nickname = null, Optional<IReadOnlyList<Snowflake>> roles = null, Optional<bool> isMuted = null, Optional<bool> isDeafened = null, CancellationToken ct = default(CancellationToken))

Remarks

The returned value will be null if the user is already in the guild.

Parameters

Name Type Description
guildID Snowflake The ID of the guild.
userID Snowflake The ID of the user to add.
accessToken string The OAuth2 access token.
nickname Optional<string> The user's new nickname.
roles Optional<IReadOnlyList<Snowflake>> The roles the user should have.
isMuted Optional<bool> Whether the user should be muted in voice channels.
isDeafened Optional<bool> Whether the user should be deafened in voice channels.
ct CancellationToken The cancellation token for this operation.

Return Value

Type Description
Task<Result<IGuildMember?>> A creation result that may or may not have succeeded.
