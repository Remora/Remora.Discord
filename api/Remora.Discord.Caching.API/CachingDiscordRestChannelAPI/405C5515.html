CreateChannelInviteAsync(Snowflake, Optional<TimeSpan>, Optional<int>, Optional<bool>, Optional<bool>, Optional<InviteTarget>, Optional<Snowflake>, Optional<Snowflake>, Optional<string>, CancellationToken) Method

Summary

Creates an invite to the given channel.
Namespace
Remora.Discord.Caching.API
Containing Type
CachingDiscordRestChannelAPI

Syntax

public override Task<Result<IInvite>> CreateChannelInviteAsync(Snowflake channelID, Optional<TimeSpan> maxAge = default(Optional<TimeSpan>), Optional<int> maxUses = default(Optional<int>), Optional<bool> isTemporary = default(Optional<bool>), Optional<bool> isUnique = default(Optional<bool>), Optional<InviteTarget> targetType = default(Optional<InviteTarget>), Optional<Snowflake> targetUserID = default(Optional<Snowflake>), Optional<Snowflake> targetApplicationID = default(Optional<Snowflake>), Optional<string> reason = default(Optional<string>), CancellationToken ct = default(CancellationToken))

Parameters

Name Type Description
channelID Snowflake The ID of the channel.
maxAge Optional<TimeSpan> The duration of the invite before expiry, or 0 for never.
maxUses Optional<int> The max number of uses, or 0 for unlimited.
isTemporary Optional<bool> Whether this invite grants temporary membership.
isUnique Optional<bool> If true, don't try to reuse an existing invite with the same settings.
targetType Optional<InviteTarget> The target type for this invite.
targetUserID Optional<Snowflake> The target user ID for this invite.
targetApplicationID Optional<Snowflake> The ID of the application to open for this invite. Required if targetType is EmbeddedApplication.
reason Optional<string> The reason to mark the action in the audit log with.
ct CancellationToken The cancellation token for this operation.

Return Value

Type Description
Task<Result<IInvite>> A creation result which may or may not have succeeded.
