AuthorizingIntegrationOwners

Gets the integrations that authorized the interaction.

Declaration
Optional<IReadOnlyDictionary<ApplicationIntegrationType, Snowflake>> AuthorizingIntegrationOwners { get; }
Property Value
Type Description
Optional<IReadOnlyDictionary<ApplicationIntegrationType, Snowflake>>
Remarks

This is a mapping of the integration type to the ID of its resource.

The dictionary contains the following, given the circumstances:
- If the integration is installed to a user, a key of UserInstallable and the value is the user ID.
- If the integration is installed to a guild, a key of GuildInstallable and the value is the guild ID. If the interaction is sent outside the context of a guild, the value is always zero.
