Interface IPermissionComputationService

Computes permission sets based on available data.

Namespace: Remora.Discord.Extensions.Services
Assembly: Remora.Discord.Extensions.dll
Syntax
[PublicAPI]
public interface IPermissionComputationService
Remarks

Note that since this service typically accesses data via the Discord API in order to gather enough information to perform its calculations, caching could result in it taking some time for the values produced by this service to reflect reality. Normally, this isn't a problem, but be aware of it and program accordingly.

ComputeMemberPermissions(Snowflake, Snowflake, Snowflake?, CancellationToken)

Declaration

Parameters
Type Name Description
Snowflake userID

The ID of the user.
Snowflake guildID

The ID of the guild the user is in.
Snowflake? channelID

The ID of the channel the user is in.
CancellationToken ct

The cancellation token for this operation.
Returns
Type Description
Task<Result<IDiscordPermissionSet>>

The computed permissions.
Remarks

channelID controls whether permission overwrite in any particular channel are taken into account. If it is omitted, the computed permissions are valid anywhere in the server where a permission overwrite does not otherwise modify the permissions.

If you want to compute effective permissions for some kind of user action, it's likely that you want to set this parameter to an appropriate value.
