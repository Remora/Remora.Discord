Returns
Type Description
Task<Result>

A gateway connection result which may or may not have succeeded.
Remarks

This task will not complete until cancelled (or faulted), maintaining the connection for the duration of it.

If the gateway client encounters a fatal problem during the execution of this task, it will return with a failed result. If a shutdown is requested, it will gracefully terminate the connection and return a successful result.
