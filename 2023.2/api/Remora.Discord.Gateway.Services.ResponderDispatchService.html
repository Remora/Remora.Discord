DispatchAsync(IPayload, CancellationToken)

Dispatches the given payload to interested responders. If the service is stopped with pending dispatches, the pending payloads will be dropped. Any payloads that have been dispatched (that is, a call to this method has returned a successful result) will be allowed to run to completion.
