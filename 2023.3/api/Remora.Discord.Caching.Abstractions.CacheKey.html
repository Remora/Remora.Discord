LocalizedStringKey(string, string)

Creates a string-based cache key from a context string and key string. The context of a key can be, for instance, a GUID or the name of the application domain that manages this key. Therefore, two keys with the same keyString can be functionally distinct by belonging to different contexts.

A new string-based cache key. This key is equal to other keys created by this method as long as context and keyString are the same.
